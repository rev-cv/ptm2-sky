
## Шаг %%%STEP%%%: Формирование мотивационного текста

Цель шага — создать краткий, вдохновляющий текст, который поможет пользователю в минуты утраты мотивации и потере смыслов осознать значимость задачи, повысит вовлечённость и мотивацию к действию.

Мотивационный текст должен быть основан на:
- выводах анализа в предыдущем шаге;
- реальных возможностях, ценностях и целях, отражённых в задаче;
- эмоциональном и практическом значении успешного выполнения;

Требования к мотивационному тексту:
- Тон — поддерживающий, но без излишнего пафоса;
- Стиль — лаконичный, без обращения ко второй персоне (не использовать «ты», «вы»);
- Объём — около 300 символов;
- Фокус — на новых возможностях, позитивных улучшениях жизни, которые станут доступными после выполнения задачи;
- Призыв к действию — скрытый, через позитивную активацию (без прямых указаний);
- Эмоциональная вовлечённость — затрагивать внутренние стремления, цели, ценности;
- Конкретность — не использовать абстрактные формулировки вроде «добьёшься успеха», а связывать мотивацию с содержанием задачи.

## Формат вывода

Верните ответ в формате JSON в котором должен содержать объект с одним единственным полем - `motivation` содержащий составленный вами ранее мотивационный текст.

Возвращайте ТОЛЬКО валидный JSON. Возвращаемая строка, должна являться валидным JSON, готовым для парсинга стандартными JSON-парсерами.

Не пишите никакого текста вне JSON-структуры.

Во пример пример правильного ответа:
```json
{
    "motivation": "Мотивационный текст …"
}
```
